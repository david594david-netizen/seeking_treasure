📻 Nostalgic Street: Treasure Hunt GameA nostalgic visual recognition game designed specifically for the elderly to stimulate cognitive function, improve visual focus, and evoke warm memories of the past.🌟 OverviewNostalgic Street (Memory Gambia Store) is an interactive web-based game where players find specific retro items within a 30-second time limit. Built with a "Vibe Coding" philosophy, it prioritizes emotional connection and accessibility over complex mechanics.✨ Key FeaturesNostalgic "Gambia Store" Vibe: Features a retro wooden shelf aesthetic with items like old TVs, rotary phones, and traditional snacks to spark reminiscence.Elder-Friendly Design:High-contrast, large-scale icons (Emoji-based).Large touch targets for users with limited dexterity.Warm, non-stressful color palette.Cognitive Training:Visual Recognition: Scanning shelves for specific items.Reaction Time: 30-second challenge to encourage mental alertness.Immersive Audio Feedback:Success Sound: A clear "Ding-Dong" chime for positive reinforcement.Error Sound: A gentle, low-pitched vibration to minimize frustration.Tada! Sound: A celebratory chime when the game ends.Modern Web Technology:Single File Architecture: All HTML, CSS, and JS are contained in one file for easy deployment.Web Audio API: Real-time synthesized sounds (no external audio files required).Responsive Design: Fully optimized for mobile, tablet, and desktop viewing using Tailwind CSS.🎮 How to PlayStart: The game begins automatically with a 30-second countdown.Objective: Look at the "Target Item" displayed at the top of the screen.Find: Search the three wooden shelves and tap/click the matching icon.Score: Each correct find increases your score. The game immediately refreshes with a new target and a new set of items.Game Over: After 30 seconds, your final score is displayed. You can tap "Play Again" to challenge your high score!🛠️ Technical DetailsFramework: Tailwind CSS for rapid, responsive UI styling.Typography: Noto Serif TC for a classic, readable look.Animation: Custom CSS Keyframes for "pop-in," "shake," and "success" effects.Sound Engine: Custom JavaScript AudioContext for synthesized feedback.🏛️ DevelopmentDeveloped by Shang-Zan Home Care (尚讚居家長照機構).Our mission is to combine technology with compassionate care, providing tools that enhance the quality of life and mental well-being of our seniors.Note: This project is part of a series of "Lively Aging" apps dedicated to senior cognitive health.
